# ProductsAPI
Clone the repository
```bash
git clone --recurse-submodules https://github.com/GediminasKripas/ProductTracker2.git
```
Change directory:
```bash
cd ProductTracker2
```

Launch docker containers:
```bash
docker-compose up -d
```
## Swagger
Url to swagger:
```bash
http://localhost:80/swagger/index.html
```
